PPContext Class¶

class opyprint.pp_context.PPContext(width=100, truncate=14, bullet='', indent='', default_bullet='- ')¶

Bases: object

Represents a pretty-printing context for constructing structured, human-readable representation of complex, composite data structures.

This context provides three groups of methods and is callable itself.

  1. The format() method format objects and returns the resulting pretty-printed string.

  2. When calling a PPContext object as a function, then the given arguments are formatted and collected. Use the flush() method to get (and clear) the collected content or the print() method to print (and clear) the collected content.

    Example:

    def describe(ppc: PPContext = None) -> str:
    ppc = ppc or PPContext()
    ppc("A foobar object, with:")
    ppc("* alpha", alpha)
    ppc("* beta", beta)
    return ppc.flush()

  3. The context managers (e.g. indent() or bullet()) temporarily adjust the pretty-print context.

Parameters

  • width (int) – Total width in characters, including bullets and indentation. Defaults to the value of the default_width class attribute.

  • truncate (int) – The truncation setting. When this value is 0, no truncation is applied. When any other positive integer value n is given, then no more than n list/tuple/set elements or dictionary items will be included and no more than n lines of a wrapped string will be included. Defaults to the value of the default_truncate class attribute.

  • bullet (str) – Optional bullet prefix string.

  • indent (str) – The indentation prefix string.

format(*args, bullet=None, style=None, key_style=None)¶

Returns a pretty-printed representation of the given arguments.

Parameters

  • args – Values to be formatted. When you provide one argument, then it will be pretty-printed normally. When you provide two arguments, then they will be formatted as a name-value pair (in a dict). When you provide three or more arguments, then these will be formatted as a list.

  • bullet (Optional[Union[str, bool]]) – When given, prefix the formatted result with a bullet, when this is not yet the case. The argument may be either a non-empty string, the first character of which is taken as the bullet, or true to use the current or default bullet.

  • style (Optional[Union[opyprint.pp_styles.PPStyles, str, int, Collection[Union[opyprint.pp_styles.PPStyles, str, int, Collection[Any]]]]]) – Optional style specifications.

  • key_style (Optional[Union[opyprint.pp_styles.PPStyles, str, int, Collection[Union[opyprint.pp_styles.PPStyles, str, int, Collection[Any]]]]]) – Optional style specifications for the key part of key-value pairs.

Return type

str

indent(indent=' ')¶

Indents the pretty-printing context.

Use this context-manager in a with statement as shown in the following example:

ppc = PPContext()
ppc("v_1")
with ppc.indent():
    ppc("v_2")
    ppc("v_3")
ppc("v_4")
ppc.print()

The above will print the following:

v1
  v2
  v3
v4
Parameters

indent (str) – The optional indentation prefix string. Defaults to the value of the static default_indent attribute.

bullets(bullet=None)¶

Prefixes each formatted object in the pretty-printing context with a bullet.

Use this context-manager in a with statement as shown in the following example:

ppc = PPContext()
ppc("v_1")
with ppc.bullet():
    ppc("v_2")
    ppc("v_3")
ppc("v_4")
ppc.print()

The above will print the following:

v1
- v2
- v3
v4
Parameters

bullet (str) – The bullet prefix string. Defaults to the value of the static default_bullet attribute or when this context is nested in another ‘bullets’ context, the bullet for that context.

print(*args, bullet=None, style=None, key_style=None, **kwargs)¶

Pretty-prints the given arguments or the pretty-printed content collected by calling the context as a function.

Additional keyword arguments are passed to the native Python ‘print’ function.

Parameters

  • args – When called without arguments the pretty-printed content collected by calling the context as a function is printed. When called with two argument and the print_name_value_pairs attributes is true, then the arguments are printed as a name-value pair (in a dict). Otherwise multiple arguments are printed as list.

  • bullet (Optional[Union[str, bool]]) – When given, prefix the formatted result with bullet, when this is not yet the case. The argument may be either a non-empty string, the first character of which is taken as the bullet, or true to use the current or default bullet. This parameter is ignored when this method is called with (other) arguments.

  • style (Optional[Union[opyprint.pp_styles.PPStyles, str, int, Collection[Union[opyprint.pp_styles.PPStyles, str, int, Collection[Any]]]]]) – Optional style specifications.

  • key_style (Optional[Union[opyprint.pp_styles.PPStyles, str, int, Collection[Union[opyprint.pp_styles.PPStyles, str, int, Collection[Any]]]]]) – Optional style specifications for the key part of key-value pairs.

Return type

None
