opyprint.utils.lt.lt(obj_1, obj_2)¶

Checks if obj_1 < obj_2.

Tries to compare the objects normally or when both objects are dictionaries, uses the dict_lt() function (which calls this function recursively).

Return type

bool

opyprint.utils.lt.dict_lt(dct_1, dct_2)¶

Checks if dct_1 < dct_2.

First tries to compare the sorted key tuples. When those are equal, tries to compare the value tuples (sorted by their keys).
