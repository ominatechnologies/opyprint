print()¶

opyprint.print.print(*args, bullet=None, end='\n', file=<colorama.ansitowin32.StreamWrapper object>, flush=False, indent='', key_style=None, style=None, truncate=14, width=100, sep=' ', **kwargs)¶

A drop-in replacement for the standard Python print function that pretty-prints the given arguments.

>>> from opyprint import print
>>> print({"alpha": [1, 2, 3], "beta": "satisfied"})
- alpha: [1, 2, 3]
- beta: satisfied
Parameters

  • args – When called with two argument and the opyprint.pp_context.print_name_value_pairs attribute is true, then the arguments are printed as a name-value pair (in a dict). Otherwise multiple arguments are printed as list.

  • bullet (Optional[str]) – When given, prefix the formatted result with a bullet, when this is not yet the case. The argument may be either a non-empty string, the first character of which is taken as the bullet, or true to use the current or default bullet.

  • end – See native ‘print’ function.

  • file – See native ‘print’ function.

  • flush – See native ‘print’ function.

  • indent (str) – The indentation prefix string.

  • key_style (Optional[Union[opyprint.pp_styles.PPStyles, str, int, Collection[Union[opyprint.pp_styles.PPStyles, str, int, Collection[Any]]]]]) – Optional style specifications for the key part of key-value pairs.

  • sep – See native ‘print’ function.

  • style (Optional[Union[opyprint.pp_styles.PPStyles, str, int, Collection[Union[opyprint.pp_styles.PPStyles, str, int, Collection[Any]]]]]) – Optional style specifications.

  • truncate (int) – The truncation setting. When this value is 0, no truncation is applied. When any other positive integer value n is given, then no more than n list/tuple/set elements or dictionary items will be included and no more than n lines of a wrapped string will be included. Defaults to the value of the PPContext.default_truncate class attribute.

  • width (int) – Total width in characters, including bullets and indentation. Defaults to the value of the ‘default_width’ class attribute of the PPContext class.

Return type

None
